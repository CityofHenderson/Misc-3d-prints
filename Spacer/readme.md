# 24.4mm Spacer

## Synopsis

Spacer to adjust for the COVID-19 faceshields avalable [here](https://github.com/Cederb/Faceshield.nu).

Dimensions are as follows:

24.4mm long

13.25mm wide

3mm wall

## Version History

v 1.0 - Public release

## Instructions

Use on American 3-hole punch on 14" (Legal) Setting.
